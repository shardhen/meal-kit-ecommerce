# Meal Kit E-commerce

A full-stack meal kit e-commerce web application built with Node.js, Express.js, EJS, MongoDB, and JavaScript.  
The application allows customers to browse meal kits, create an account, log in, add products to a shopping cart, and place orders. It also includes a clerk/admin role for managing meal kit products.

## Key Features

- User registration and login
- Password validation and encrypted password comparison
- Session-based authentication
- Role-based access for customers and clerks
- Meal kit product listing
- Shopping cart functionality
- Add, remove, and update cart items
- Order placement with confirmation email
- Clerk/admin product management
- Add, edit, and delete meal kits
- Image upload for meal kit products
- MongoDB database integration
- Error handling for unauthorized access and missing pages

## Project Highlights

- Built an Express.js server with modular routing and EJS layouts.
- Connected the application to MongoDB using Mongoose.
- Implemented customer and clerk roles to separate shopping and product management features.
- Created shopping cart logic using session storage.
- Integrated Mailgun to send welcome and order confirmation emails.
- Added product management features, including image upload, edit, and delete functionality.
- Deployed the application using Vercel.
